# Калькулятор на Python

Это мой первый учебный проект — графический калькулятор, созданный с помощью библиотеки tkinter.

## Функционал
- Базовые арифметические операции
- Графический интерфейс

## Как запустить
1. Убедитесь, что у вас установлен Python.
2. Скачайте файл PythonApplication1.py.
3. Запустите его командой: python PythonApplication1.py

## Стек технологий
- Python 3
- tkinter
